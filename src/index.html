Welcome to KENNEDY'S Restaurant Management API

Welcome to the Restaurant Management API. This API allows you to manage various aspects of a restaurant including users, cities, states, addresses, categories, restaurants, menu items, comments, drivers, orders, and more.

Want to get all users? Please redirect to /api/users

Want to get details about a specific city? Please redirect to /api/cities

Need information about states? Please redirect to /api/states

Want to manage addresses? Please redirect to /api/addresses

Interested in categories? Please redirect to /api/categories

Looking for restaurant details? Please redirect to /api/restaurants

Managing menu items? Please redirect to /api/menu-items

Want to read comments? Please redirect to /api/comments

Need driver information? Please redirect to /api/drivers

Managing orders? Please redirect to /api/orders

Interested in order menu items? Please redirect to /api/order-menu-items

Restaurant owners' details? Please redirect to /api/owners

Want to manage the catalogue? Please redirect to /api/catalogue

Looking for order statuses? Please redirect to /api/order-status

Need to register or log in? Please redirect to /api/auth/register or /api/auth/login
